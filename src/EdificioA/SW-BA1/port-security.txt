! =====================================================================
! SW-BA1  —  Port Security nas portas de acesso (SRS — tarefa do Rodrigo)
! 1 MAC sticky por porta, violacao = shutdown.
! =====================================================================
! NOTA: estes switches de acesso BA1 sao do Romeu (SW-BA1-A1_Norte,
! _Centro, _Sul, etc.). Aplicar SO nas portas de ACESSO (com PCs/
! telefones) — NUNCA nos trunks/uplinks. Ajustar o range abaixo para
! as portas access reais de cada switch.
! =====================================================================

SW-BA1-A1 (aluno e1/1, prof e1/0, telefones e1/2 e1/3):
conf t
interface Ethernet1/0
switchport mode access
switchport access vlan 20
interface Ethernet1/1
switchport mode access
switchport access vlan 10
interface Ethernet1/2
switchport mode access
switchport access vlan 70
interface Ethernet1/3
switchport mode access
switchport access vlan 70
interface range Ethernet1/0 - 3
switchport port-security
switchport port-security maximum 1
switchport port-security mac-address sticky
switchport port-security violation shutdown
end
SW-BA1-A2 (alunos e1/1 e1/2, informática e1/0, telefone e1/3):
conf t
interface Ethernet1/0
switchport mode access
switchport access vlan 50
interface range Ethernet1/1 - 2
switchport mode access
switchport access vlan 10
interface Ethernet1/3
switchport mode access
switchport access vlan 70
interface range Ethernet1/0 - 3

end

SW-BA1-Dist (guests e1/1 e2/0, impressora e1/2, telefone e1/3):
Atenção aqui — as portas de convidados (e1/1, e2/0) vão levar isolamento (protected ports / private VLAN) numa fase posterior. Port Security combina com isso, mas para não criares conflito agora, incluo-as na mesma. A VLAN dos convidados é a 60.
conf t
interface Ethernet1/1
switchport mode access
switchport access vlan 60
interface Ethernet2/0
switchport mode access
switchport access vlan 60
interface Ethernet1/2
switchport mode access
switchport access vlan 80
interface Ethernet1/3
switchport mode access
switchport access vlan 70
interface range Ethernet1/1 - 3
switchport port-security
switchport port-security maximum 1
switchport port-security mac-address sticky
switchport port-security violation shutdown
interface Ethernet2/0
switchport port-security
switchport port-security maximum 1
switchport port-security mac-address sticky
switchport port-security violation shutdown
end
